코딩덕이 특별한 이유

오리가 물 위를 우아하게 헤엄치듯, 코딩덕은 여러분이
IT 세계를 자연스럽고 즐겁게 헤쳐나갈 수 있도록 함께합니다.

🦆
왜 이름이 코딩덕일까요?

개발자들 사이에는 "러버덕 디버깅(Rubber Duck Debugging)"이라는 비법이 있습니다. 막힌 코드를 해결할 때 책상 위 오리 인형에게 코드를 소리 내어 설명하다 보면, 스스로 문제를 발견하게 되는 거죠.

코딩덕은 여러분의 그 오리가 되겠습니다. 막혀도 같이 생각하고, 달릴 때는 더 빨리 달릴 수 있도록.

🎯

취업 중심 커리큘럼

현업에서 실제로 쓰이는 기술 스택 중심으로 설계된 커리큘럼으로, 수료 후 바로 실무에 투입될 수 있는 역량을 키웁니다.

🚀

실전 프로젝트

배운 내용을 실제 프로젝트에 적용하며 포트폴리오를 쌓아가는 방식으로, 이론과 실전을 동시에 잡습니다.

☕

백엔드 전문 강사진

Java · Spring · JPA · AWS를 현업에서 직접 다루는 백엔드 개발자 출신 강사들이 실무 노하우를 그대로 전달합니다.

🤝

커뮤니티 지원

수강 후에도 지속적인 멘토링과 동문 네트워크를 통해 성장을 이어갈 수 있도록 지원합니다.

AI 시대의 개발자

개발자가 사라진다고요?
오히려 지금이 기회입니다.

취업 준비생들이 가장 많이 하는 걱정, "AI가 개발자를 대체하면 어떡하지?"
결론부터 말씀드리면, AI는 개발자를 없애는 게 아니라 역할을 바꿉니다.

기존 개발자
  • 📝 코드를 한 줄 한 줄 직접 작성
  • 🔧 기능 구현 자체에 시간의 대부분 소비
  • 📚 API 문서·보일러플레이트 반복 작성
  • 🐛 버그를 찾고 수정하는 데 집중
🤖
AI 도입
AI 시대 개발자
  • 🏗️ 시스템 설계·아키텍처에 집중
  • 🔍 AI 생성 코드를 검증·개선·최적화
  • 🎯 AI에게 정확한 요구사항 전달
  • ⚡ 더 빠르게 더 많은 서비스 구축
01

AI가 개발 속도를 높이면

더 많은 서비스가 만들어지고, 그 서비스를 운영·유지·확장할 더 많은 개발자가 필요해집니다.

02

AI 코드는 검증이 필요합니다

AI가 짠 코드가 왜 동작하는지 모르면 장애가 났을 때 손댈 수 없습니다. 기초가 있는 개발자가 핵심입니다.

03

'잘하는 개발자' 수요가 폭발 중

단순 코더는 줄지만, 아키텍처를 이해하고 AI를 활용하는 개발자의 연봉과 수요는 동시에 오르고 있습니다.

위의 역할 변화, 코딩덕 커리큘럼이 정확히 커버합니다.

코딩덕 강의 과정

백엔드 취업에 필요한 모든 기술을 단계별로 완주합니다.

취업 준비생
🦆

프로 코딩덕

비전공자도 시작할 수 있는 백엔드 풀코스.
Java 기초부터 AWS 배포까지, 취업에 필요한 모든 것을 4개월 안에 완성합니다.

1
Java 기초 & 자료구조 · 알고리즘

코딩 테스트와 기술 면접의 핵심 영역. 배열·스택·큐·트리·그래프·해시 등 CS 핵심 자료구조를 Java로 직접 구현하고, 시간 복잡도 분석부터 최적화까지 훈련합니다.

Java 8+ OOP 자료구조 알고리즘 코딩 테스트 대비
🎯 기술 면접 핵심 📝 코딩 테스트 직결
  • ✅ LeetCode Easy~Medium 레벨 혼자 풀기
  • ✅ 기술 면접에서 "자료구조란?"에 자신 있게 답변
  • ✅ 시간 복잡도 O(n) vs O(log n) 즉시 계산
  • ✅ 네이버·카카오 코딩 테스트 1차 통과 수준 도달
2
Spring Boot & JPA 백엔드 개발

국내 백엔드 시장 표준 스택. RESTful API 설계부터 JPA로 DB를 다루는 방법까지 실습합니다. QueryDSL로 타입 안전한 쿼리를 작성하고, Spring Security + JWT로 인증·인가를 직접 구현합니다.

Spring Boot JPA / Hibernate QueryDSL Spring Security JWT REST API MySQL Swagger
  • ✅ Spring Boot REST API 서버 혼자 구축
  • ✅ JWT 인증·인가 시스템 처음부터 구현
  • ✅ 면접에서 "JPA N+1 문제"를 원리부터 해결책까지 설명
  • ✅ 실제 서비스 수준의 백엔드 API 설계 및 문서화
3
현대 백엔드 기술 스택

현업에서 실제로 사용하는 기술들을 체험합니다. Redis로 캐싱·세션 처리를 구현하고, 메시지 큐 개념으로 비동기 처리를 이해합니다. Kotlin과 Java를 비교하며 현대적인 코드 스타일도 익힙니다.

Redis Kafka 개념 Kotlin 입문 비동기 처리 캐싱 전략 API 설계 패턴
  • ✅ Redis 캐싱으로 API 응답 속도 대폭 개선
  • ✅ 비동기 이벤트 처리 아키텍처 설계 이해
  • ✅ Kotlin으로 더 간결하고 안전한 코드 작성
  • ✅ "Redis를 왜 쓰나요?" 면접 질문에 실전 경험으로 답변
4
AWS 클라우드 배포 & DevOps

EC2·RDS·S3로 실제 서비스를 배포하고, Docker로 컨테이너 환경을 구성합니다. GitHub Actions CI/CD 파이프라인을 직접 구축해 자동 빌드·배포를 경험합니다.

AWS (EC2 · RDS · S3) Docker GitHub Actions CI/CD Linux 기초
  • ✅ 내가 만든 서비스를 실제 URL로 접속 가능하게 배포
  • ✅ GitHub에 push하면 자동으로 서버 반영되는 CI/CD 구축
  • ✅ Docker로 "내 PC에서만 된다"는 문제 완전 해소
  • ✅ 포트폴리오에 "직접 운영 중인 서비스" URL 첨부 가능
5
팀 프로젝트 & 취업 코칭

실제 팀 프로젝트로 완성도 높은 포트폴리오를 만듭니다. 이력서 피드백, 모의 기술 면접, 코딩 테스트 집중 훈련으로 최종 취업까지 함께합니다.

팀 프로젝트 코드 리뷰 기술 면접 대비 이력서 피드백 포트폴리오 완성
💼 취업 집중 코칭
  • ✅ 배포된 실서비스 포트폴리오 GitHub 공개
  • ✅ 모의 기술 면접에서 30분 이상 막힘 없이 답변
  • ✅ 이력서 한 줄에 실제 숫자("트래픽 N배 개선") 기입
  • ✅ IT 기업 서류·코딩 테스트·기술 면접 전 과정 준비 완료
🕐 주 5회 · 4개월 과정
👥 소수 정예 운영
💼 백엔드 취업 집중 코칭
🧩
자료구조 & 알고리즘은 기술 면접의 핵심입니다

네이버·카카오·라인 등 국내 주요 IT 기업의 기술 면접과 코딩 테스트에서 자료구조·알고리즘은 필수 관문입니다. 코딩덕 프로 과정은 취업 실전에 맞춘 문제 풀이 훈련을 커리큘럼에 포함합니다.
